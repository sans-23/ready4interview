## Exercise 1: Traffic Light


**Problem:** Create a `TrafficLight` enum where each light has a color (RED, YELLOW, GREEN), a duration in seconds, and a `next()` method that returns the next light in the cycle (RED -&gt; GREEN -&gt; YELLOW -&gt; RED).

**Requirements:**

- Each light has a `duration` property: RED = 30s, YELLOW = 5s, GREEN = 25s
- `next()` method returns the next `TrafficLight` in the cycle
- `display()` method prints the color and duration
